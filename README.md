# Request Tasks

**React + TypeScript + Vite + Chakra UI**

## Система управления заявками для аптек с современным UI и строгой типизацией.

## Функционал формы заявки

- [x] **Поиск и выбор аптеки**
- [x] **Приоритеты**
- [x] **SLA**
- [x] **Гарантийный статус** (чекбокс)
- [x] **Drag & Drop файлов** (PDF/изображения с превью)
- [x] **Валидация всех полей** (обязательные поля + кастомная логика)
- [x] **Анимации hover/focus/scale** (Chakra UI + CSS transitions)
- [x] **Responsive дизайн** (Mobile-first + Flex/Grid)

## Быстрый старт

```bash
npm install
npm run dev
```
